# 传奇 H5 ARPG · 主线任务设计细化（v2）

> 基于现有数据驱动引擎（`quest_defs.json` + `quest.js`）的细则化方案。
> 目标：在已落地的 6 幕 / 12 章骨架之上，补齐**剧情推进节奏、触发条件与完成目标、关键剧情节点与转折、成长与奖励分层**，确保主线逻辑连贯、层次清晰、沉浸感与推进动力兼顾，杜绝任务断裂与目标模糊。

---

## 一、当前框架盘点

| 层 | 现状 | 本次强化点 |
|---|---|---|
| 引擎 | 状态机 `locked→available→active→completed`，事件总线 `onKill/onEnterMap/collect/talk`，HUD 追踪栏 | 新增**章节感知对话**（同一 NPC 按当前激活章节切换台词，承载节点与转折） |
| 数据 | 6 幕 `arc0~arc5`、每幕 2 章、目标类型 `reach/kill/collect/talk/boss/explore` | 补全 `theme/story/twist/pacingNote/rewardTier` 元数据 + 每章 `flavor` 沉浸短句 |
| 叙事 | NPC 仅 2 句通用台词，节奏高度同质（每幕=清怪+打Boss） | 6 位 NPC 多拍台词、统一的四象核心悬念线、3 处反转 |
| 断层风险 | `arc0_ch2`/`arc1_ch2` 的 `reach/explore` 目标指向「玩家已所在地图」→ 上一章结束瞬间自动完成，章节被压缩为空壳 | **已修复**：交接章改为「对话下一 NPC / 拾取信物」等真实步骤 |

## 二、主线叙事总纲（One-Line Spine）

> **「渊墟觉醒」** —— 身负修炼系统印记的觉醒者，穿越五张主题地图集齐**四象核心（地/火/冰/灵）**，终战被裂隙腐化的初代守渊人化身·远古巨龙，以自身印记为锁**永久封印裂隙**。
> 「无敌觉醒」不是变强，而是**选择守护**。

- **核心矛盾**：封印（守渊人职责）↔ 裂隙（腐化诱惑）↔ 集核者自身被注视、被反噬。
- **四象核心设定**：地之核（尸王）、火之核（魔君）、冰之核（巨魔）、灵之核（渊/英灵）。每枚都是「被腐化的守渊人」的残响。
- **巨龙真相（终幕反转）**：远古巨龙 = 初代守渊人的集体化身，被裂隙腐化后成为威胁；击败它并非消灭，而是**解救**。
- **渊的身份悬念（第四幕铺设）**：龙裔先知·渊是初代守渊人后裔，集四核究竟为「封印」还是「重开裂隙」——直到终幕才揭晓他为盟友。

## 四、逐幕细化（五维度全覆盖）

### 幕0 · 觉醒印记（比奇城郊 · L1）
- **剧情前提**：觉醒者自外界坠入渊墟，遇引路人玄机子。
- **推进节奏**：平缓教学 → 首杀小高潮 → 真相交接（3 个目标 + 1 交接目标）。
- **触发条件**：进入游戏即 `arc0_ch1` 可用（无前置）；`arc0_ch2` 前置 `arc0_ch1` 完成。
- **完成目标**：`reach 比奇城郊` + `kill slime×3` + `talk 玄机子`（教学）→ `talk 玄机子`（觉醒真相）+ `talk 阿蛮`（接守村预警）。
- **关键节点与转折**：①玄机子揭示「印记=守渊人烙印」；②阿蛮点破「兽潮只是裂隙异动第一道涟漪」——埋下终局悬念。
- **成长与奖励**：`exp 100 / gold 200 / skillPoints 1 / unlock tutorial_done`。对应**新手奠基**：学会操作、解锁技能点。

### 幕1 · 原野告急（比奇城郊 · L1）
- **剧情前提**：哥布林王率兽潮围村。
- **推进节奏**：紧接序章的小高潮，战斗密度上升，Boss 双阶段。
- **触发条件**：`arc0` 完成；等级 ≥1（已满足）。
- **完成目标**：`talk 阿蛮`（警报）+ `kill goblin×8` + `boss goblinKing(2阶)` → `talk 阿蛮`（谢礼）+ `collect 守村信物×1`。
- **关键节点与转折**：守村战胜利后，残魂借阿蛮传话——枯骨回廊藏着核心碎片②，主线正式指向洞窟。
- **成长与奖励**：`exp 200 / gold 300 / skillPoints 1 / unlock coreFragmentClue`。**新手初成**：首次多阶段 Boss 实战。

> **修复说明**：原 `arc1_ch2_o2` 为 `explore 比奇城郊`（玩家已在该图）→ 进入即自动完成，章节被压成「一句话」。改为 `collect 守村信物`，增加一次真实探索拾取。

### 幕2 · 枯骨低语（僵尸洞窟 · L5）
- **剧情前提**：深入枯骨回廊，碑灵残魂指引。
- **推进节奏**：**转折幕**——叙事密度高于战斗，首个「真相初显」节点。
- **触发条件**：`arc1` 完成；等级 ≥5（需前期升级达标，引擎 `canStart` 卡等级门）。
- **完成目标**：`reach 僵尸洞窟` + `kill skeleton×12` + `collect 碑文残片×3` → `boss corpseKing(2阶)` + `talk 残魂`（领地核②）。
- **关键节点与转折**：尸王死前低语「四核归一，巨龙将醒」——主角开始怀疑「集核」的真正目的，叙事从「打怪」升维为「解谜」。
- **成长与奖励**：`exp 500 / gold 500 / coreFragment +1 / unlockAdvancement 1`。**成长期·觉醒①**：解锁职业二转，第一次获得核心碎片（地之核）。

### 幕3 · 焚心试炼（烈焰火山 · L10）
- **剧情前提**：熔狱试炼，兽人部族盘踞火山口。
- **推进节奏**：中高潮，三阶段 Boss，情感张力 + 自我怀疑。
- **触发条件**：`arc2` 完成；等级 ≥10。
- **完成目标**：`reach 烈焰火山` + `kill orc×15` → `boss flameLord(3阶)` + `talk 炽`（领火核③）。
- **关键节点与转折**：魔君本是守渊人之一，被裂隙腐化；**主角意识到自身印记也可能反噬**——将外部威胁内化为角色焦虑。
- **成长与奖励**：`exp 700 / gold 600 / coreFragment +1`。**精进期·专精**：火之核入手，强化「你也可能是下一个被腐化者」的叙事压强。

### 幕4 · 霜原誓约（冰封雪原 · L16）
- **剧情前提**：冰原结盟，冰狼群与冰霜巨魔。
- **推进节奏**：情感高潮 + 悬念铺设，为终幕反转蓄力。
- **触发条件**：`arc3` 完成；等级 ≥16。
- **完成目标**：`reach 冰封雪原` + `kill iceWolf×12` → `boss frostTroll(2阶)` + `talk 霜`（立誓领冰核④）。
- **关键节点与转折**：霜长老立誓时抛问——**「龙裔先知·渊集四核，究竟为封印还是重开裂隙？」** 制造对终局盟友的信任危机。
- **成长与奖励**：`exp 900 / gold 700 / coreFragment +1 / unlockAdvancement 2`。**誓约期·觉醒②**：终极觉醒，四核将齐，抉择逼近。

### 幕5 · 深渊终焉（魔龙深渊 · L22）
- **剧情前提**：四核已集，深渊裂隙将开。
- **推进节奏**：终极高潮 → 反转结局 → 尾声升华，节奏收束。
- **触发条件**：`arc4` 完成；等级 ≥22。
- **完成目标**：`reach 魔龙深渊` + `collect 英灵核心×4` → `boss dragon(3阶)` + `talk 渊`（封印裂隙）。
- **关键节点与转折（双反转）**：①巨龙非敌，而是被腐化的初代守渊人化身；②结局选择「以印记为锁、永久封印」而非「夺取力量」——**无敌觉醒 = 守护而非征服**。
- **成长与奖励**：`exp 2000 / gold 2000 / skillPoints 3 / ending 无敌觉醒`。**封印期·结局**：通关，解锁结局与后续内容钩子。

## 五、节奏曲线设计（Pacing Curve）

```
情感强度
 高 │                                    ╱▔▔╲ (幕5 终战+反转)
    │                          ╱▔▔╲ (幕4 誓约悬念)
    │                ╱▔▔╲ (幕3 魔君+自我怀疑)
    │          ╱▔▔╲ (幕2 尸王+真相初显)
    │    ╱▔╲ (幕1 首Boss)        ╱ (幕0 教学)
 低 │___╱                                          ──→ 进度
      序章   幕1    幕2    幕3    幕4    幕5
```

- **章节数/目标类型变化**：每幕固定 2 章（行动章 + 交接章），但目标组合刻意错落——`reach/kill/collect/talk/boss` 轮换，避免「每幕都是清怪+Boss」的疲劳。
- **节奏张弛**：幕0~1 偏战斗直觉建立；幕2 起叙事占比抬升；幕3~4 情感压强递增；幕5 收束于反转与升华。
- **Boss 阶段梯度**：2阶（哥布林王/尸王/巨魔）→ 3阶（魔君/巨龙），难度与叙事重量同步爬升。

## 六、触发条件与防断裂机制

| 机制 | 实现 | 防断裂作用 |
|---|---|---|
| 前置链 | `prereqArc` + `prereqChapter` 串联 | 严格线性，无跳跃空洞 |
| 等级门 | `arc.minLevel` + `canStart` 校验 | 高幕不因越级提前解锁而失衡 |
| 自动开启 | 章完成即 `refreshStates` + 启最早 `available` 章 | 无「卡住不知下一步」 |
| 地图自动完成 | `enterMap` 时 `reach/explore` 即时判定 | 避免「到了地图却不触发」 |
| 对话兜底 | 每 NPC 含 `default` 台词 + HUD 始终显示 NPC 名 | 避免「目标模糊/不知找谁」 |
| 退化章节修复 | 交接章改为真实步骤（对话下一 NPC / 拾取信物） | 杜绝「上一章结束瞬间自动完成」 |

> 本次实测发现并修复：`arc0_ch2`（原 `reach 比奇城郊`）、`arc1_ch2`（原 `explore 比奇城郊`）因目标地图=当前地图，进入即自动完成，章节被压缩为空壳。已改为 `talk 阿蛮` + `collect 守村信物`，并补放 `守村信物` 采集物。

## 七、成长与奖励分层机制

主线奖励与叙事里程碑**强绑定**，让「变强」本身成为剧情反馈：

| 档位 | 触发幕 | 奖励构成 | 叙事含义 |
|---|---|---|---|
| 新手期·奠基 | 幕0 | `skillPoints 1` + 基础金币 | 学会生存 |
| 新手期·初成 | 幕1 | `skillPoints 1` + 线索解锁 | 初历实战 |
| 成长期·觉醒① | 幕2 | `coreFragment +1` + `advancement 1` | **职业二转**，获地之核 |
| 精进期·专精 | 幕3 | `coreFragment +1` | 火之核，自我怀疑压强 |
| 誓约期·觉醒② | 幕4 | `coreFragment +1` + `advancement 2` | **终极觉醒**，四核将齐 |
| 封印期·无敌觉醒 | 幕5 | `skillPoints 3` + `ending` | 守护结局，通关 |

- `coreFragment` 累计驱动终幕开启（集齐 4 枚解锁魔龙深渊传送），把「收集进度」变成可见的叙事资产。
- `advancement` 解锁二转/终极觉醒，把「数值成长」包装为「守渊人阶位晋升」。

## 九、转折设计清单（具体反转点）

1. **幕0→幕1**：兽潮不是偶然，是裂隙异动前兆（悬念启动）。
2. **幕2 尸王之死**：「四核归一，巨龙将醒」——集核目的遭质疑。
3. **幕3 魔君之死**：守渊人也会被腐化 → 主角自身印记可能反噬（内化焦虑）。
4. **幕4 霜长老立誓**：对盟友「渊」的动机抛出信任危机（最大悬念）。
5. **幕5 双反转**：巨龙=被腐化守渊人；结局=守护而非夺力（主题落点）。

## 十、已落地的代码改动（本次实施）

| 文件 | 改动 | 作用 |
|---|---|---|
| `quest_defs.json` | 每幕加 `theme/story/twist/pacingNote/rewardTier`；每章加 `flavor` | 设计元数据入数据，HUD 显示沉浸短句 |

## 十一、微信小游戏 / 小程序部署兼容说明

- 本设计**数据驱动**（JSON + 事件总线），与渲染解耦，H5 与原生小游戏可复用同一份 `quest_defs.json`。
- 端口注意（属 B 组小程序收尾任务 #31/#33，非本次范围）：
  - `quest.js` 的 `fetch('quest_defs.json')` 在小游戏需改为 `wx.request` 或打包进包；
  - HUD 追踪栏 / NPC 对话依赖 `document` DOM，小游戏需改为 canvas 绘制或 wx 组件。
- 建议：主线数据层先抽离为纯 JSON，UI 层按平台分别实现（H5 用 DOM、小游戏用 canvas），最大化复用叙事内容。

## 十二、后续可扩展点

1. ** escorte / 护送目标**：当前 schema 已支持 `escort`，可在幕4 加一段「护送霜长老过冰原」丰富节奏。
2. **分支抉择**：幕4 的「信任危机」可做成真二选一（影响结局文案），提升重玩价值。
3. **结局变体**：依据是否集齐全部核心碎片 / 是否触发隐藏对话，给出「守护 / 夺力 / 残缺」三结局。
4. **周常/世界任务**：主线之外接一层轻量日常，维持长线留存。
