## 2. 发现的 Bug 及修复

### Bug 1: 三个模块导出大小写问题 ❌→✅

**现象**: `window.Mount`, `window.Destructible`, `window.BodyPart` 为 `undefined`，坐骑/可破坏物/部位破坏面板无法渲染

**根因**: Agent 生成的模块使用 `window.mount`(小写)/`window.destructible`/`window.bodyPart` 导出，但 UI 代码检查 `typeof Mount/Destructible/BodyPart`(大写)

**修复**: 在 3 个模块的 window 导出处增加了大写别名

**涉及文件**:
- `mount.js`: +`window.Mount = exports;`
- `destructible.js`: +`window.Destructible = exports;`
- `body_part.js`: +`window.BodyPart = exports;`

### Bug 2: 核心面板套装显示异常 ❌→✅

**现象**: 套装效果显示为 `套装效果 ([object Object]/4)`

**根因**: `renderCorePanel()` 中 `const setLv = CoreSystem.checkSetBonus(loadout)` 返回的是对象 `{level, desc, bonuses}` 而非数字，直接拼接字符串导致 `[object Object]`

**修复**: `index.html` 第4274行改为解构对象取 `.level` 属性

```js
// 修复前
const setLv = CoreSystem.checkSetBonus(loadout);
// 修复后
const setResult = CoreSystem.checkSetBonus(loadout);
const setLv = setResult.level;
```
